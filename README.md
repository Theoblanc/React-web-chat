# React-web-chat
<br />

made React-web-chat program (react + node.js + graphql + apollo)


<br />

**Technology Stack**

| Component | Technology |
|---|:---:|
| `Frontend` | 	React 16+ | 
| `Backend` | node.js, grapql |
| `Security` | JWTtoken |
| `Auth` | Facebook, Local |
| `Database` | Mysql, Prisma |
| `Client Build	` | npm, yarn, webpack, babel |

## LOGIN 방법 

1. 로그인을 한다.
2. email로 secret키가 감.
3. 확인 후 입력하면 JWT TOKEN 발행 (TOKEN은 1시간)
