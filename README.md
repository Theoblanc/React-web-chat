# React-web-chat
<br />

> made React-web-chat program (react + node.js + graphql + apollo)

<br />

**Technology Stack**

| Component | Technology |
|---|:---:|
| `Frontend` | 	React 16+ | 
| `Backend` | node.js, grapql |
| `Security` | JWTtoken |
| `Auth` | Facebook, Local |
| `Database` | Mysql, Prisma |
| `Client Build	` | npm, yarn, webpack, babel |
